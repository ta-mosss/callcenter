Getting Started
Setup Guide
1
Create a Twilio Account
Sign up at twilio.com (free trial available). After verifying your email, go to the Console Dashboard and note your Account SID and Auth Token. You'll need these in your .env file.
2
Get a South African Phone Number
In Twilio Console → Phone Numbers → Buy a Number → Search for South Africa (+27).

Option A — New number: Buy a new +27 number from Twilio (~R25/month).
Option B — Port your 087 number: Go to Console → Phone Numbers → Port & Host → submit a porting request for 087 093 1138. This takes 2–4 weeks and keeps your existing number.
3
Configure Your .env File
Copy .env.example to .env and fill in:
# .env
TWILIO_ACCOUNT_SID=ACxxxxxxxxxxxxxxxxxxxx
TWILIO_AUTH_TOKEN=your_auth_token
TWILIO_PHONE_NUMBER=+27870931138
FORWARD_TO=+27737335518
ADMIN_PASSWORD=YourSecurePassword
PORT=3000
4
Deploy the Server
Install dependencies and start the server:
npm install
npm start
Recommended hosting (free):
• Railway.app — push to GitHub, connect, done. Free tier included.
• Render.com — free Node.js web service.
• Fly.io — generous free tier.

For local testing, use ngrok:
npx ngrok http 3000
5
Configure Twilio Voice Webhooks
In Twilio Console → Phone Numbers → Your number → Voice Configuration:

Set A call comes in → Webhook → HTTP POST:
https://YOUR-DOMAIN.com/voice/inbound
Set Call Status Changes:
https://YOUR-DOMAIN.com/voice/status
6
Test Your Setup
Call your Twilio number. You should hear:

"Thank you for calling Mbulaheni Group Integrated Solutions. Please hold while I try to connect you to the next available agent."

Then your cellphone (+27 73 733 5518) will ring. If you don't answer within 30 seconds, callers are sent to voicemail.
7
Verify Your Forwarding Number (Trial Accounts)
On a free Twilio trial, you can only call verified numbers. Go to Console → Phone Numbers → Verified Caller IDs and add +27 73 733 5518. Twilio will call or SMS it to verify. Paid accounts can call any number without verification.
